# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability, please report it privately.

- Preferred: open a GitHub Security Advisory ("Report a vulnerability")
- Alternative: contact the repository owner

Please include:

- Affected version/commit
- Reproduction steps
- Impact assessment
- Any suggested mitigation

## Scope

This project provides a P2P transport/session library. Threat modeling and correct usage depends on the embedding application.
